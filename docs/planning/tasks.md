# 项目任务清单

## 项目名称

PURE-JADE：面向情绪支持场景的共情策略生成原型系统

## 当前阶段

协议设计与原型规划阶段。

当前最重要的目标是：

1. 冻结四张卡片的 JSON Schema；
2. 完成研究内容二的共情策略决策逻辑；
3. 明确 ESConv 的使用路径；
4. 再进入最小可运行 Demo 实现。
